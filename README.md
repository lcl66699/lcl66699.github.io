## 介绍

基于vuepress的个人博客

⭐ 求个小星星 😆


### 插件

- [vuepress-plugin-qrocde](https://github.com/openHacking/vuepress-plugin-qrcode) 展示当前网址二维码供移动设备扫描

## 用法


安装依赖

```sh
cd vuepress-blog
yarn # 或者npm i
```



打包项目

```sh
npm run docs:build
```

结果会在`docs/.vuepress/`目录下生成一个`dist`文件夹，里面就是打包后的代码
